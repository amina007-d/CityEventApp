## Full Description

CityEvent is a modern Android application that helps users discover and explore upcoming events in their city quickly and conveniently.

The app allows users to browse concerts, festivals, exhibitions, sports events, and other public activities in one place with a clean and user-friendly interface.

### Key Features

- Browse upcoming events sorted by date
- Search events by name
- Filter events by category
- View detailed event information (date, time, venue, address)
- Save events to Favorites
- Register and log in securely
- View and manage your user profile

CityEvent integrates with external event APIs to provide up-to-date event information and uses Firebase services for authentication and user data management.

The application is designed with modern Android architecture principles and ensures smooth performance and intuitive navigation.

## Privacy Notes

CityEvent uses Firebase Authentication and Firestore to manage user accounts and personalized data.

### Data Collected

- Email address (for account authentication)
- User-generated data (favorites, comments)

### Data Usage

- Account authentication
- Personalization (favorites and user-specific content)

### Data Sharing

- The app does not sell user data.
- The app does not share personal data with third parties.

### Security

- All data is transmitted securely over HTTPS.
- Authentication is handled securely via Firebase.

CityEvent does not collect sensitive personal information, precise location data, or financial information.
