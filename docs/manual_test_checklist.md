# Manual Test Checklist (>= 10)
1. Launch app first time (no crash)
2. Sign up with valid email/password
3. Sign in with existing account
4. Wrong password shows friendly error
5. Feed loads events (loading -> list)
6. Pagination: scroll to bottom, next page loads
7. Turn off internet: feed shows cached data
8. Open details screen for an event
9. Add/remove favorite; persists after restart
10. Open comments: add comment, see it immediately
11. Edit comment; realtime updates
12. Delete comment; realtime updates
